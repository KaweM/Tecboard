## 📌 Sobre

Projeto feito com HTML e CSS, com foco em um layout limpo, direto e responsivo.
A proposta é apresentar uma página inicial com uma mensagem clara e um botão de ação.

---

## 🛠️ Tecnologias

* HTML5
* CSS3

## ▶️ Como usar

Clone o repositório:

```bash

cd Tecboard
```

Abra o arquivo `index.html` no navegador.
